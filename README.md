# 🐾 **Pet Profile QR – Identificación Digital para Mascotas**

<img src="/info/img1.png" alt="Pet Profile QR" />

<p>
Pet Profile QR es un sistema moderno de identificación para mascotas mediante un <strong>código QR</strong> que enlaza a un <strong>perfil web</strong> con toda la información necesaria para que cualquiera pueda ayudarte a recuperar a tu mascota en caso de extravío.
</p>

<p>
Este proyecto es una versión piloto del sistema, donde se diseñan perfiles personalizados para cada mascota, incluyendo fotos, información médica, datos del dueño y un enlace QR listo para imprimir o colocar en un llavero/placa.
</p>

---

## ✨ **Características principales**

</div>

<div align="center">

<ul align="center">
  <li>Perfil web accesible desde cualquier celular.</li>
  <li>No usa GPS ni rastreo.</li>
  <li>No requiere instalar aplicaciones.</li>
  <li>Código QR único para cada mascota.</li>
  <li>Información clara, ordenada y fácil de leer.</li>
  <li>Incluye datos de la mascota, vacunas, alergias, galería y datos del dueño.</li>
</ul>

---

## 📲 **¿Cómo funciona Pet Profile QR?**

</div>

<div align="center">

<ol align="center">
  <li>Se crea un perfil personalizado para la mascota.</li>
  <li>Se genera un código QR único.</li>
  <li>El dueño puede imprimirlo o colocarlo en una placa/llavero.</li>
  <li>Al escanearlo, se muestra información completa: nombre, fotos, alergias, vacunas, teléfono, redes, ubicación.</li>
  <li>Si la mascota se pierde, quien la encuentre puede contactarte al instante.</li>
</ol>

# 🐾 **Pet Profile QR – Identificación Digital para Mascotas**

Pet Profile QR es un sistema moderno de identificación para mascotas mediante un **código QR** que enlaza a un **perfil web** con toda la información necesaria para que cualquiera pueda ayudarte a recuperar a tu mascota en caso de extravío.

Este proyecto es una versión piloto del sistema, donde se diseñan perfiles personalizados para cada mascota, incluyendo fotos, información médica, datos del dueño y un enlace QR listo para imprimir o colocar en un llavero/placa.

---

## ✨ **Características principales**

- Perfil web accesible desde cualquier celular.  
- No usa GPS ni rastreo.  
- No requiere instalar aplicaciones.  
- Código QR único para cada mascota.  
- Información clara, ordenada y fácil de leer.  
- Incluye:  
  - Datos de la mascota (edad, especie, raza, características).  
  - Información médica (vacunas, alergias).  
  - Galería de fotos.  
  - Datos de contacto del dueño (teléfono, redes, ubicación).  

---

## 📲 **¿Cómo funciona Pet Profile QR?**

1. Se crea un perfil personalizado para la mascota.  
2. Se genera un código QR único.  
3. El dueño puede imprimirlo o colocarlo en una placa/llavero.  
4. Al escanear el QR, la persona verá:  
   - Nombre de la mascota  
   - Fotos  
   - Información importante (alergias, vacunas)  
   - Teléfono del dueño  
   - Redes sociales  
   - Ubicación aproximada  
5. Si la mascota se pierde, quien la encuentre puede contactarte al instante.
